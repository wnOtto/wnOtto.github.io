背景
不论TCP、UDP，亦或长连接、短连接的网络游戏与单机游戏的本质区别在于网络数据的传输（HTTP、Socket）。今天，使用Unity来实现一个最最简单Socket的网络案例！
工具：Visual Studio、Unity
语言：C#

一、案例基本步骤

服务器：
第一步：创建一个Socket对象

Socket 对象名 = IPAddress.Parse(寻找地址，套接字类型，协议方式);

第二步：绑定监听消息IP和端口号

IPAddress ip = IPAddress.Parse(“ip地址”);
EndPoint endPoint = new IPEndPoint(ip,端口号);
socket对象.Bind(endPoint);

第三步：监听客户端的连接请求

设置处理最大请求数 tcpScoket.Listen(100);
暂停当前线程，监听接收到客户端发来的连接请求；
当接收到客户端的连接请求后，在本地服务器创建一个socket与客户端连接，并返回出来Socket socket = tcpScoket.Accept();

第四步：向客户端发送消息
