Font Awesome之于Hexo上的应用（二）

发表于 2020-05-07 | 次阅读

  本篇主要讲述在Hexo中Font Awesome 4是如何去使用的，正如前篇说到的，我的博客就是使用Hexo来生成的，自己也是在学以致用的状态中。。。。。。
  相信对此篇感兴趣的小伙伴，已经有使用过Hexo啦（了解Hexo），没有用过的话也不影响，来吧，开搞！。

准备工作

1.使用Hexo创建项目并完成主题配置（我配置的是next主题，便以其为例，其他主题参照即可！）
2.简单了解Hexo、next、MarkDown、HTML、YAML（相信这对于你不是问题，可忽略）

应用实例

  Hexo中使用的便是Font Awesome，Hexo在创建项目过程中就包含有经典的Font Awesome 4最新文件了。也就是说，我们只用关心如何使用它即可。
  相信看过前一篇的小伙伴已经会在Html中使用Font Awesome了。然而，我们的Hexo中是可以不去直接使用html的，而是在其YAML配置文件和MarkDown标记文件中使用！

  1. Font Awesome文件目录
    目录位置：项目的public/lib/font-awesome
    css引用位置：项目的public/lib/font-awesome/css/font-awesome.css
  2. _config.yml主题配置文件
    文件位置：项目的/themes/next/_config.yml
    YAML:配置文件用于数据结构配置，后缀.yml，注释使用#，空格使用严格，大小写严格
    使用FontAwesome方式：|| 图标名
    使用FontAwesome两处位置：
  3. markdown文件
    文件创建位置：项目的source/_posts/
    文件创建方式：手动新建文件名.md文件 或 GitBash命令创建：$ hexo new page 文件名
    文件作用：写我们博客文章，自动解析为html。
    使用方式如下：直接按照其在html中的嵌入写法使用 具体效果的使用：请参照前一篇的具体使用
