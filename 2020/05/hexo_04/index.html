Hexo使用七牛云存储资源

发表于 2020-05-19 | 更新于: 2020-05-20 | | 次阅读

前言

就我的github个人站点而言，因为其是属于国外站点，所以国内访问时，其网站本身加载就有点慢，更别说存放在了该站点下的一些图片、音乐、视频等资源的加载显示了。
国内的服务站点加载自己根目录的资源时，效果就会好的多，但如果国外访问，同样又会变差！ 如何避免这种较差的体验效果呢！
就拿我的个人站点举例吧！要想国内访问时拥有其资源快速加载显示的效果。
方式一：直接使用国内其他网站的资源链接（但大多网站设置了防盗链，虽然可以破解，但不推荐）
方式二：自己申请注册个资源空间、然后创建个国内服务（或国外服务），使用此空间外链加载资源！

七牛云

使用七牛云进行资源存储，写文章时直接引用七牛云的图片等外链，利用七牛云强大的融合cdn服务可以加速图片等资源的加载！
实名注册后免费获取10G存储空间！可使用七牛云空间基本的资源管理操作！
hexo拥有七牛云插件，使用插件更加方便了其资源的上传与使用！

牛云注册与配置

首先，到七牛云官网创建一个账号且实名认证，这个过程很简单，这里就不再赘述。
进入七牛云主页，选择对象存储模块进行新建空间，存储空间名称只要满足要求随便填，存储区域默认即可，但要注意访问控制一项必须选择公开空间，否则就不能用外链的形式进行访问了。

点击创建确定创建后，在融合 CDN 测试域名一栏中可以看到系统分配的融合 CDN 测试域名，不过这个域名只有30天的期限，所以最好还是使用自己的域名。如果要是使用自己的域名的话，需要配置一下。由于这不是本文的重点，所以有这个需求的小伙伴可以参考七牛云文档——域名接入

上传图片文件以后，复制外链连接就可以利用这个链接访问这个图片了。

工具使用

如果每次都需要像上面一样上传图片的操作就比较麻烦了，在这里可以使用一些工具来方便的上传，比如MPic与PicGo，使用工具可以直接将图片拖进去就可以使用外链连接了，操作十分简便。对此工具感兴趣的可以自行百度下载安装与了解使用，这不做过多介绍，主要介绍下面hexo的七牛云插件。
