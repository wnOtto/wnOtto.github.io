Hexo基础之常用命令详解

发表于 2020-05-13 | | 次阅读

Hexo 框架可以帮助我们快速创建一个属于自己的博客网站，那么熟悉 Hexo 框架提供的命令有利于我们管理博客！

1.hexo init命令

描述：初始化本地文件夹为网站的根目录

2.hexo new命令

描述：新建.md文章，一般可以简写为 hexo n，可嵌套-p -i选项

3.hexo generate命令

描述：生成静态文件，一般可以简写为 hexo g，可嵌套-d选项名表示生成后部署

4.hexo deploy命令

描述：部署网站，一般可以简写为 hexo d，可嵌套-g选项同样表示生成后部署

5.hexo server命令

描述：启动本地服务器，一般可以简写为 hexo s，可嵌套-p、-i、-s选项

说明：有个hexo-server 插件 $ npm install hexo-server --save（需不需要安装记不清了）

6.hexo clean命令

描述：清理缓存文件，是一个比较常用的命令

7.hexo –save命令

描述：hexo插件安装

8.hexo –safe命令

描述：表示安全模式，用于禁用加载插件和脚本

9.hexo –debug命令

描述：表示调试模式，用于将消息详细记录到终端和 debug.log 文件

10.hexo –silent命令

描述：表示静默模式，用于静默输出到终端

11.&&

描述：表示命令相继执行
