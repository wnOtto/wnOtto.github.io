Hexo添加评论系统

发表于 2020-05-14 | 更新于: 2020-05-14 | | 次阅读

Hexo 在Next主题下配置Valine评论系统，三步轻松搞定！

为什么选择Valine，我觉得我的需求就是方便，简洁，最重要一点是要支持Markdown ，对比之下，最终选定了这款Valine，当然，如果你不喜欢这样简洁的，可以参考 Hexo（NexT 主题）评论系统哪个好？

第一步，注册LeanClound，获取APP ID 和 APP Key

Valine 是基于 LeanCloud官网作为数据存储的，所以需要注册一个账号，注册完成后，我们找到创建应用

创建完成后，打开应用，点击左边的存储，查看是否有Comment和 Counter，如没有则创建，权限默认设为所有用户。创建完成后，如下图：

说明：如没有创建，那么在显示 post 页面时，请求 LeanCloud 的 api 接口，可能会返回 error code=101

点击设置 → 安全中心 ,将除了数据存储的服务全部关闭，并绑定web安全域名，防止恶意调用。

最后点击 应用 Keys，等一会过来复制 AppID 和 Appkey。
