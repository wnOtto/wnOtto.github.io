Font Awesome 是什么（一）

发表于 2020-05-07

  Font Awesome是我在打造自己博客的学习过程中而接触到一个网站流行图标工具，故而我觉得相的相当有必要介绍一下。

首先大家来看一下下面几个图标，是不是有点眼熟呢，没错就是我的博客左侧菜单栏上的图标！

        
那么，这些图标是怎么来的？又该如何使用的呢？还有没有其他的小图标使用吗？这就是我要介绍Font Awesome的起因！
1. Font Awesome 简介

Font Awesome是一套绝佳的图标字体库和CSS框架！

使用CSS提供的特性可以对字体图标进行大小、颜色、旋转、阴影、组合等效果的更改！
一个字库，675个矢量图标，可无限缩放，无需依赖javaScript，完全免费，支持商用，亦可用于桌面系统！
图标不断添加，包括经典Font Awesome4 以及最新Font Awesome5

官方网站：网站有许多，推荐下面两个

国内(直观浏览图标及名称)：https://fontawesome.dashgame.com/
国外(更全/图标搜索/打开网站较慢)：https://fontawesome.com/icons?d=gallery&q=th
