前言

就我的github个人站点而言，因为其是属于国外站点，所以国内访问时，其网站本身加载就有点慢，更别说存放在了该站点下的一些图片、音乐、视频等资源的加载显示了。
国内的服务站点加载自己根目录的资源时，效果就会好的多，但如果国外访问，同样又会变差！ 如何避免这种较差的体验效果呢！
就拿我的个人站点举例吧！要想国内访问时拥有其资源快速加载显示的效果。
方式一：直接使用国内其他网站的资源链接（但大多网站设置了防盗链，虽然可以破解，但不推荐）
方式二：自己申请注册个资源空间、然后创建个国内服务（或国外服务），使用此空间外链加载资源！

七牛云

使用七牛云进行资源存储，写文章时直接引用七牛云的图片等外链，利用七牛云强大的融合cdn服务可以加速图片等资源的加载！
实名注册后免费获取10G存储空间！可使用七牛云空间基本的资源管理操作！
hexo拥有七牛云插件，使用插件更加方便了其资源的上传与使用！

牛云注册与配置

首先，到七牛云官网创建一个账号且实名认证，这个过程很简单，这里就不再赘述。
进入七牛云主页，选择对象存储模块进行新建空间，存储空间名称只要满足要求随便填，存储区域默认即可，但要注意访问控制一项必须选择公开空间，否则就不能用外链的形式进行访问了。


点击创建确定创建后，在融合 CDN 测试域名一栏中可以看到系统分配的融合 CDN 测试域名，不过这个域名只有30天的期限，所以最好还是使用自己的域名。如果要是使用自己的域名的话，需要配置一下。由于这不是本文的重点，所以有这个需求的小伙伴可以参考七牛云文档——域名接入

上传图片文件以后，复制外链连接就可以利用这个链接访问这个图片了。

工具使用

如果每次都需要像上面一样上传图片的操作就比较麻烦了，在这里可以使用一些工具来方便的上传，比如MPic与PicGo，使用工具可以直接将图片拖进去就可以使用外链连接了，操作十分简便。对此工具感兴趣的可以自行百度下载安装与了解使用，这不做过多介绍，主要介绍下面hexo的七牛云插件。

添加插件与配置

进入hexo根目录安装七牛云的插件，命令行输入：npm install hexo-qiniu-sync --save安装插件
在站点配置文件_config.yml文件中加入以下几行配置：

qiniu:
  offline: false
  sync: true
  bucket: wnotto               # 在七牛云中新建的存储空间的名称
  access_key: your access_key  # 七牛云密钥管理中的AK
  secret_key: your secret_key  # 七牛云密钥管理中的SK
  dirPrefix: qny            # 上传的资源子目录前缀.若设置，需与urlPrefix同步
  urlPrefix: http://p2zukkwm9.bkt.clouddn.com/qny    # 外链前缀
  up_host: http://upload.qiniu.com    # 使用默认配置即可
  local_dir: cdn       # 本地目录
  update_exist: true   # 是否更新已经上传过的文件(仅文件大小不同或在上次上传后进行更新的才会重新上传)
  image: 
    folder: img
    extend:
  js:
    folder: js
  css:
    folder: css
# dirPrefix 字段为上传的资源子目录前缀。如果不写的话，默认为static；如果设置的话，需要与 urlPrefix同步;
# urlPrefix字段为外链链接的前缀，这里要注意把dirPrefix加进来；
# local_dir为本地目录，image字段中folder为本地图片的存储路径，extend代表对图片进行附加的变换，这里先不填。
# js、css等也可以作为资源上传，甚至是音乐、视频等，直接在后面添加即可

在hexo根目录下新建文件夹cdn，并且在cdn文件夹下新建文件夹img、js等(上面配置的这必须都添加上)。命令行如下：(当然也可手动创建这些文件夹)
$ mkdir cdn
$ cd ./cdn
$ mkdir img
$ mkdir js
$ mkdir css

使用图片

如果要在博文中引用图片的话
+ 本地图片存放
需要将图片存放至/cdn/img文件夹中(可以继续添加子文件夹)
+ 图片上传至七牛云
执行下面命令任意一个，如果检查到图片有更新，会将其上传到配置的七牛云空间

hexo s  或  server
或
hexo g  或  
或
hexo qiniu sync 或 s 或 sync2 或 s2
